## 解决什么问题

Hermes 在早期踩过的坑：
- README 越写越厚，AI 反而更迷路
- 术语不统一，团队沟通成本极高
- 项目状态全靠人脑记忆，AI 接新项目就像空降兵

## 使用方式

拿去改造成你自己的。路径、术语、规则都可以调整，核心是**先有规范，再谈规模**。
